# Ultraenv Next.js Example

This example demonstrates how to integrate ultraenv with a Next.js application.

## Setup

```bash
npm install next react react-dom ultraenv
```

## Running

```bash
npm run dev
```

## What it does

1. Provides type-safe environment variables for both server and client components
2. Validates environment variables at build time
3. Automatically generates TypeScript types for `process.env`
